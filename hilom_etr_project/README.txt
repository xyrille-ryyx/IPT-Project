HILOM - Teen Mental Health Analytics Dashboard

How to run:
1. Open terminal inside this folder.
2. Install libraries: pip install -r requirements.txt
3. Run: python app.py
4. Open: http://127.0.0.1:5000

Default admin login:
Username: admin
Password: admin123

Main requirements included:
- Flask login page
- Data visualization dashboard
- CSV preprocessing using Pandas
- OOP classes for data processing, analytics, and chart generation
- Matplotlib charts saved in static/charts
- Account management page
- Learnings/realizations and assigned task table
- Basic error handling for 404, 500, and general exceptions
